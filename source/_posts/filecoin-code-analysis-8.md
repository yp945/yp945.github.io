```
    [functions]
    // 实例化检索矿工
    // 设置处理免费检索的handle方法：handleRetrievePieceForFree
   +NewMiner(nd minerNode) : *Miner

▼-minerNode : interface
    [methods]
   +Host() : host.Host
   +SectorBuilder() : sectorbuilder.SectorBuilder
```
